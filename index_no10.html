no 10 - CSS

CSS stands for "Cascading Style Sheets".

It is a style sheet language to style an HTML document.

CSS describes how the content of HTML should be displayed in the web page.
You can add style such as fonts, colors, size and spacing of your contents.

With CSS you can also create and change the layout of your web pages.
